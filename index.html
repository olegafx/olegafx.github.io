Это случилось!

Уже очень давно собирался завести себе блог, где можно постить мысли длиннее, чем 140 символов в Твиттере.

Есть очень много штук, которые на чужих сервисах-то нормально и не расскажешь. Поэтому, сделал свой без фатального недостатка.

Сейчас пошла мода на flat-cms. Если кратко, то это CMS без баз данных. Всё основано на обычных файлах. В основе этого блога лежит Jekyll — генератор сайтов с Github Pages. В качестве основы для темы выбрал hikari. Сам блог хостится на бесплатном GitHub Pages, упомянутом выше.

Jekyll — очень крутая штука. Markdown, командная строка, git. Всё, как я люблю. Есть несколько моментов, которые реализуются через костыли, но во всём остальном очень даже.

Буду писать сюда про всякие интересные штуки, которые есть сейчас в мире разработки и не только.

Надеюсь, вам понравится. До скорых встреч!
