CloudUp — сервис мгновенного обмена данными (+ бонус внутри)

Есть огромное количество сервисов, позволяющих делать скриншоты и сразу же загружать их в облако. Скриншот можно посмотреть, зайдя по ссылке. То есть не надо ничего сохранять к себе на компьютер, мучаться с пересылкой по Skype или чему-то еще. Сделали скриншот, отправили ссылку. Люди сами посмотрят его в удобное для них время.

Например, есть LightShot. Можно настроить Dropbox, чтобы он делал нечто подобное. Платный droplr умеет отправлять не только скриншоты, но и разные файлы — архивы, презентации, айдио-файлы и т.д.

CloudUp

На горизонте появился еще один подобный сервис — CloudUp. В чем его особенности:

  • можно пользоваться совершенно бесплатно и загружать до 1000 файлов по 200 мегабайт каждый
  • загрузка всех основных типов документов
  • начать просмотр по ссылке можно еще до того, как файл полностью загружен
  • одним из разработчиков является Guillermo Rauch — создатель socket.io. Кстати, не забудьте посмотреть новый пример чата в master-ветке socket.io (вот и пример видео на сервисе CloudUp)
  • сервис и приложение, как я понимаю, написаны на NodeJS

Если вы цените скорость обмена данными и удобство этого дела, то советую попробовать данный сервис. Есть приложение для Windows, Mac. Веб-версия настолько крута, что ей можно спокойно пользоваться на телефоне.

Бонусы

Проект еще находится в стадии закрытого тестирования. У меня есть 10 инвайтов. Можете зарегистрироваться по этой ссылке.

Кстати, про сам сервис я узнал из этого замечательного доклада. Всем советую посмотреть и вдохновиться.
